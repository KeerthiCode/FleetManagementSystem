# Fleet Management System

## Overview

Fleet Management System is a web-based application developed using Spring Boot, MySQL, HTML, CSS, and JavaScript. The system helps organizations manage drivers, vehicles, and delivery operations through a centralized dashboard.

## Project Objective

The main objective of this project is to simplify fleet operations by maintaining driver information, vehicle details, and delivery tasks in a single system.

## Features

### Driver Management

* Add Driver Details
* View Driver Information
* Store Contact Information
* Store License Details

### Vehicle Management

* Add Vehicle Details
* View Vehicle Information
* Track Vehicle Availability Status

### Delivery Task Management

* Create Delivery Tasks
* Track Delivery Status
* Manage Source and Destination Locations

### Dashboard

* View Drivers
* View Vehicles
* View Delivery Tasks
* User-Friendly Interface
